Manual


Main window




It is divided into left sidebar and right container. You can navigate through tabs by clicking on the appropriate page link on the sidebar. The tabs are organized into: main page (current page), single molecule, batch molecules, about.


Running calculations


Single molecule

1) click on a link single molecule you will be switched to the single molecule tab (please see figure below)



2) Enter molecule in SMILES format in SMILES input area and press Enter.

3) Select appropriate temperature by moving Temperature slider

4) Predictions are refreshed upon change of SMILES input or Temperature slider

5) Read out a prediction from results of prediction area

6) Figure 2D structure from SMILES and table Calculated molecular descriptors have only informative nature


Batch molecules


Input file

If you wish to calculate OH constant for a set of molecules, you will need to prepare an input file. The best way is to use your favorite spreadsheet editor. Name columns as following: SMILES and Temp. In SMILES you have to provide structure of molecule in a SMILES format, and in Temp you need to provide temperature reaction. Next, you have to save the spreadsheet as a CSV file (comma separated file). Please see a figure below:




Calculations




1) Click on Browse files button and navigate to the input file

2) After selecting CSV file the calculation is done automatically

3) When the calculation process is done, you should see a table with the results (please see a figure below)




4) You can download the whole table by scrolling at the bottom and clicking on a Download data as CSV button
